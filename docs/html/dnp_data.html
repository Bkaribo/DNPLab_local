class dnpLab.dnpData.dnpData(data=array([], dtype=float64), axes=[], axesLabels=[], params={}, procList=[])¶

dnpData Class for handling dnp data

The dnpData class is inspired by pyspecdata nddata object which handles n-dimensional data, axes, and other relevant information together.

This class is designed to handle data and axes together so that performing NMR processing can be performed easily.

Attributes: data (numpy.ndarray): Numpy Array containing data axes (list): List of numpy arrays containing axes of data axesLabels (list): List of axes labels for data params (dict): Dictionary of parameters for data
