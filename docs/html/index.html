Welcome to odnpLab's documentation!¶

odnpLab is an open source python package for importing and processing Overhauser dynamic nuclear polarization (ODNP) data.

  • Version: 1.0
  • Current Release: 1.0.0
  • Documentation Build Date: 04/01/2020, 13:18:56
  • author: Bridge12 + Han Lab

Features¶

  • Import Bruker, Varian, Kea Data
  • NMR Processing
  • Extract Hydration Dynamics Information
  • Create Publication Quality Figures
