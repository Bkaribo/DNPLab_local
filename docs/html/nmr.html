dnpLab.dnpNMR.align(allData, procParameters)¶

Alignment of NMR spectra down given axes dimension

dnpLab.dnpNMR.fourierTransform(allData, procParameters)¶

Perform Fourier Transform down axes dimension given in procParameters

Note

Assumes dt = t[1] - t[0]

Parameters

  • allData (dnpData, dict) -- Data container

  • procParameters (dict, procParam) -- Processing parameters

Returns

Processed data in container

Return type

allData (dnpData, dict)

Example:

procParameters['axes'] = 't'
procParameters['zeroFillFactor'] = 2
procParameters['shift'] = True
procParameters['convert2ppm'] = True

allData = dnpLab.dnpNMR.fourierTransform(allData, procParameters)
dnpLab.dnpNMR.integrate(allData, procParameters)¶

Integrate data down given dimension

Parameters

  • allData (dnpData,dict) -- Data container

  • procParameters (dict, procParam) --

    Processing Parameters axes_label: str

    dimension to integrate

    integrateCenter:

    center for width of integration

    integrateWidth:

    width of integration window

Returns

Processed data

Return type

allData (dnpData,dict)

Exampe: .. code-block:: python

procParameters = {

'axes' : 't', 'integrateCenter' : 0, 'integrateWidth' : 100, }

dnpLab.dnpNMR.integrate(allData,procParameters)

dnpLab.dnpNMR.removeOffset(allData, procParameters)¶

Remove DC offset from FID by averaging the last few data points and subtracting the average

Parameters

  • allData (dnpData, dict) -- Data container for data

  • procParameters (dict,procParam) -- Processing Parameters

procParameters['axes'] = 't'
procParameters['offsetPoints'] = 10

outData = dnpLab.dnpNMR.removeOffset(allData,procParameters)
Returns

allData (dnpData, dict)

dnpLab.dnpNMR.window(allData, procParameters)¶

Apply Apodization to data down given dimension

Parameters

  • allData (dnpData, dict) -- data container

  • procParameters (dict, procParam) -- parameter values

Note

Axis units assumed to be seconds

Exampe:

procParameters = {
        'linewidth' : 10,
        'axes' : 't',
        }
allData = dnpLab.dnpNMR.window(allData,procParameters)
